# MDX Cheatsheet

MDX combines Markdown and JSX, allowing you to write JSX in your Markdown documents.

## Basic Markdown Syntax

- **Headers**: `# H1`, `## H2`, `### H3`, etc.
- **Bold**: `**bold**`
- **Italic**: `*italic*`
- **Links**: `[title](https://)`
- **Images**: `![alt](image-url)`
- **Lists**:
  - Unordered: `- item`
  - Ordered: `1. item`
- **Code**: `` `code` `` for inline, ```` ```language \n code block \n``` ```` for blocks
- **Quotes**: `> quote`
